# Sources

Papers referenced in this project. Download via DOI links below.

## Geological classification
Catani, F., Lagomarsino, D., Segoni, S., Tofani, V. (2013).
Landslide susceptibility estimation by random forests technique.
*Natural Hazards and Earth System Sciences*, 13, 2815–2831.
https://doi.org/10.5194/nhess-13-2815-2013

tldr: a study that reports how sensible landslide susceptibility assessments are to how geological information is grouped, and no standard methods are available. the lithology approach ranked second best, the genetic/process-based approach ranked first. so i guess we're gonna make it work.
